# Reaction-speed

🎮 Reaction Speed Arena

This project is a simple browser-based game built to test and improve reaction speed. The idea is straightforward: shapes appear on the screen, and the player has to click them as fast as possible to record reaction times.

This is still an early version of the project and is mainly being used as a testing ground for different ideas and gameplay mechanics. A lot of things are still under development, and the current version is not the final one. The goal is to keep improving the experience based on testing and feedback.

🚧 Current Status
The game is functional but still in a testing phase
Some features are experimental and may change
UI and animations are still being refined
Scoring and difficulty systems are being adjusted

🎯 Features (Work in Progress)
Random shape spawning
Reaction time tracking
Difficulty levels (still being tuned)
